# Training Plan Template

This file is optional athlete-specific context.

Use it only when it has been filled with a real user's goals, constraints, and training structure.
If it is still generic or template-like, treat plan context as limited and do not invent athlete-specific goals from it.

## What to put here

Add only user-specific context such as:

- current training goals
- current sport focus
- known constraints or injury considerations
- typical weekly structure
- key workout types
- practical execution preferences
